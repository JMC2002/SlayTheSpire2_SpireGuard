# 🧾 Changelog

所有对本项目的重要更改都将记录在此文件中。

版本号规则： 主版本号.次版本号.修订号，其中主版本号涉及到大功能完善，次版本号原则上Steam创意工坊发布新版本后进行更新，修订号每次涉及代码的commit后更新（从0开始）。

## [1.1.0] - 2026-05-15
### Fixed
移除大厅阶段的联机状态缓存补丁，避免在双方都启用 SpireGuard 时触碰进房握手链路。

### Changed
- 自己作为客机时默认不再禁止本机控制台，改为由“客机时禁止本机控制台”设置项单独控制。

### Added
新增“检测其他玩家控制台”设置项，检测到其他玩家的控制台网络动作时可弹出提示框。

## [1.0.5] - 2026-05-08
### Added
主机拦截客机控制台网络动作时，可通过 JmcModLib 原生单按钮弹窗提示被拦截的玩家名称。

新增“主机拦截时弹出提示”设置项，可关闭拦截弹窗。

## [1.0.4] - 2026-05-08
### Changed
发布 manifest 标记为非玩法影响，便于作为本地联机保护工具使用。

## [1.0.3] - 2026-05-08
### Added
新增联机客机控制台保护：客机无法主动打开或提交控制台命令，主机端会拒收客机发来的控制台网络动作。

新增 JmcModLib 设置项，可启用或停用 SpireGuard 保护逻辑。

修正 README 与发布 manifest 中的占位和其他 MOD 残留描述。
