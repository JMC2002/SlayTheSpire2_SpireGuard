# 🧾 Changelog

所有对本项目的重要更改都将记录在此文件中。

版本号规则： 主版本号.次版本号.修订号，其中主版本号涉及到大功能完善，次版本号原则上Steam创意工坊发布新版本后进行更新，修订号每次涉及代码的commit后更新（从0开始）。

## [1.0.4] - 2026-05-08
### Changed
发布 manifest 标记为非玩法影响，便于作为本地联机保护工具使用。

## [1.0.3] - 2026-05-08
### Added
新增联机客机控制台保护：客机无法主动打开或提交控制台命令，主机端会拒收客机发来的控制台网络动作。

新增 JmcModLib 设置项，可启用或停用 SpireGuard 保护逻辑。

修正 README 与发布 manifest 中的占位和其他 MOD 残留描述。
