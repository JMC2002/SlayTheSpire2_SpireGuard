[h1]SpireGuard (Disable Console in Multiplayer)[/h1]
[h2] 1. Overview [/h2]
Do your teammates keep secretly opening the console in multiplayer? Use this MOD to solve it! (Only your local machine needs to install it.)

[url=https://github.com/JMC2002/SlayTheSpire2_SpireGuard] GitHub Repository [/url]
[url=https://www.bilibili.com/video/BV1CXdPBZE8D] Demo Video (Bilibili) [/url]

[h2] 2. Features [/h2]
[list]
    [*] When the host enables this feature, if a client uses a console command or a MOD based on console commands, the host will intercept it and show a popup telling the host which player used which command.
    [img]https://images.steamusercontent.com/ugc/11871054468909674251/D1E338B443217A4BF9774CAF510914A8ABEACF00/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]
    [*] As a client, you cannot do anything about interception, but you can monitor who sent which command
    [img]https://images.steamusercontent.com/ugc/16399244011958331106/B571D821779F4AA871919ABFD959F17086FE885A/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]
    [*] The feature can be enabled or disabled in settings
    [img]https://images.steamusercontent.com/ugc/12196734696829028346/5BF894441CB1F71309D8AC3ADEFF4030C5F2FA38/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]
[/list]

[h2] 3. Notes [/h2]
[list]
    [*] Because of the game's mechanics, installing any MOD for the first time can cause save desynchronization. You can use [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3747533700] BetterSaveSlots [/url] to solve this
    [*] This MOD requires the following dependency:  
• [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3747526103] JmcModLib [/url]  
    [*] The localization content of this MOD (including the workshop page and changelog), except Chinese, is generated by ChatGPT. It currently supports all languages officially supported by the game. If you think any wording is inaccurate or would like to contribute your own translation, feel free to open an issue / PR on GitHub or contact me directly
[/list]

[h2] 4. Compatibility [/h2]
[list]
    [*] This mod makes extensive use of reflection and may stop working after game updates. If you find that the mod no longer works, please report it promptly.
[/list]
